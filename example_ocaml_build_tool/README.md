# OCaml build tool examples

## ocamlc

### Build

```
cd ocamlc_build
make build
```

### Clean

```
cd ocamlc_build
make clean
```


## ocamlbuild

### Build

```
cd ocamlbuild_native
make build
```

### Clean

```
cd ocamlbuild_native
make clean
```

## OMake

### Build

```
cd omake_build
omake
```

### Clean

```
cd omake_build
omake clean
```
